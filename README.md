# ED_colab
Repositorio para la actividad grupal del módulo de Entornos de Desarrollo

# Objetivos de la actividad
<ul>
  <li>Desarrollar habilidades prácticas en el uso de GitHub para la colaboración mediante forks, ramas y PRs.</li>
  <li>Comprender el rol del mantenedor y del contribuidor en un proyecto compartido.</li>
  <li>Aprender la capacidad de documentar de forma correcta los cambios realizados en una Pull Request.</li>
  <li>Practicar la revisión y aprobación de contribuciones.</li>
  <li>Aprender a manejar conflictos y mantener repositorios sincronizados entre un fork y el repositorio original.</li>
  <li>Promover un flujo de trabajo organizado.</li>
</ul>
